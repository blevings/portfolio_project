HTML Tags - Basic Page Structure

Tag Tag Name Tag Description
Document type declaration <!DOCTYPE html> Indicates the document is an HTML document
HTML tag <html> Creates the structure for the HTML content
Head tag <head> Contains the metadata for a webpage
Page Title <title> Holds the title of the webpage. Placed inside the <head> tag
Body tag <body> Contains the content visible on the webpage
Header <header> Container for navigational links or introductory content
Main <main> Used to encapsilate the dominant content within a webpage
Footer <footer> Used to encapsilate contact info, copyright info, site map, etc.
Paragraph <p> Used to contain a block of text.
div <div> Used to define a division or section in the HTML document
